Escaped Character

What is Escaped Character?
Some characters have special meaning in regular expressions and must be escaped. All escaped characters begin with the \ character.
Within a character set, only \, -, and ] need to be escaped.
Escaped Character Regex
\000 \xFF \uFFFF \cI
\t \n \v \f
\r \0 \. \\
\+ \* \? \^
\$ \[ \] \{
\} \( \) \|{
\/
Octal Escape
Octal escaped character in the form \000. Value must be less than 255 (\377).
Hexadecimal Escape
Hexadecimal escaped character in the form \xFF.
Unicode Escape
Unicode escaped character in the form \uFFFF.
Control Character Escape
Escaped control character in the form \cZ. This can range from \cA (NULL, char code 0) to \cZ (EM, char code 25).
