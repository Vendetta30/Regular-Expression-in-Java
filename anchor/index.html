Anchor

What is Anchor?
Anchors are unique in that they match a position within a string, not a character.
Anchor Regex
SubexpressionMatches
^ Matches any single character except newline. Using m option allows it to match newline as well.
$ Matches word characters.
\b Matches nonword characters.
\B A character set that can be used to match any character, including line breaks.
Begining
Matches the beginning of the string, or the beginning of a line if the multiline flag (m) is enabled. This matches a position, not a character.
Match Any
Matches the end of the string, or the end of a line if the multiline flag (m) is enabled. This matches a position, not a character.
Word
Matches a word boundary position such as whitespace, punctuation, or the start/end of the string. This matches a position, not a character.
Not Word
Matches any position that is not a word boundary. This matches a position, not a character.
